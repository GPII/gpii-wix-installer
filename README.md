gpii-wix-installer
==================

WiX based Windows Installer for the GPII

Requirements for building GPII for Windows
------------------------------------------
- Git
- Node.js v.0.10.x
- Grunt
- Visual Studio 2013
- Inno Setup

Additional requirements for building the installer
-------------------------------------------------
- Wix
- MSBuild Extension Pack

Building
--------
cd setup  
build
